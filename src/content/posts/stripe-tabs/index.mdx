## Creating Stripe's Blog Tab Section

In this article, we'll explore how to recreate Stripe's beautiful tab interface that features a sliding highlight effect.

The component uses React hooks with useRef and useEffect to calculate the position of the active tab and create a dynamic clip-path that highlights just that tab.

I created this component after completing the course of one of my favourite design engineers out there - [Emil Kowalsky](https://emilkowal.ski)
